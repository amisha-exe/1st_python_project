# 🔐 Secret Code Encoder & Decoder

A fun Python project that can **encode** and **decode** secret messages!

## ✨ How it works
- Enter **1** to **encode** a message
- Enter **2** to **decode** a message

## 🔒 Encoding
- Words with **more than 3 letters** get random characters added around them
- Words with **3 or less letters** get reversed

## 🔓 Decoding
- Reverses the encoding to get back the original message

## 🛠️ Built with
- Python
- `random` module
- `string` module
